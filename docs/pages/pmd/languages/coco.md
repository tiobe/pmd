The [Coco](https://cocotec.io/) language is a modern programming language designed
specifically for building event-driven software.

## Support in PMD
Starting from version 6.48.0, Coco support was added to CPD.

### Limitations
- Support for Coco only extends to CPD to detect code duplication in Coco source files.
